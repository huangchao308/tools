# tools

## Usage

Get how to use this tool by running:

```bash
make build
./bin/tools i18n -h
```

## Examples

```bash
./bin/tools i18n php -f /Users/murphy/Desktop/App土耳其语翻译.xlsx -s 服务端 --include.language tr --old.file /Users/murphy/Documents/workspace/php/oversea-server/app/messages/tr.php
```
